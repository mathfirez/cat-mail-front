What is Catmail?

Catmail is the result of a side project that allows you to send messages which will be printed in my small portable printer that has a cat face on it. The beautiful diagram below displays how it works:


Basically, when the message is sent, it gets processed and stored until the client requests for it. The client is designed to fetch for messages individually periodically. Try sending me a message below, it will (hopefully) get printed whenever my client is running.
